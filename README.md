## Install
1. Install Golang:
    - [golang.org](https://golang.org/dl/)
2. Install git .
3. ``` cd ~  ```
4. ``` git clone https://github.com/nightimer-AU/Vegas.git  ```
5. ``` cd Vegas/  ```
6.  ``` go run vega.go  ```
